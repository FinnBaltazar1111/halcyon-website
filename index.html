Defog Your Chromebook

E-Halcyon is a bypass for "The Fog," which is Google's mitigation for the unenrollment and downgrading of Chrome OS devices. It allows you to completely bypass enterprise enrollment on managed Chromebooks, even if it is on a version where SH1MMER is patched.

RecoMod is a Chrome OS recovery image modification toolkit allowing for arbitrary code execution in verified mode. It provides the user with utilities that are useful for debugging and repair.

A Use Case for Everyone

Students

E-Halcyon can be used as a way to bypass Google's SH1MMER mitigation, allowing you to once again use your managed Chromebook as if it were never enrolled. It also doesn't modify the device's internal storage, and simply removing the drive E-Halcyon is installed on will wipe any trace of it from the device.

Hobbyists

Want to screw around with Chrome OS without needing developer mode? E-Halcyon and RecoMod provide an easy way do do that. It also serves as an alternative to MrChromebox's fixflags images in case you can no longer boot Chrome OS off of the internal storage.

Sysadmins

E-Halcyon, as well as the associated RecoMod tool, contain many utilities which are useful for the diagnosis and repair of Chrome OS devices. It can also fix certain issues with the device's firmware which may otherwise prevent Chrome OS from booting.
